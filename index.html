Proyecto: Incubadora con Termostato

Diseño y construcción de una incubadora casera con control automático de temperatura.

📦 Descripción

Esta incubadora está hecha con una caja de icopor para conservar el calor, un sistema eléctrico con termostato y sensores de temperatura. Su diseño permite mantener un ambiente estable, ideal para incubar huevos o realizar proyectos biológicos.

🎯 Objetivo del proyecto

El objetivo es construir una incubadora funcional, económica y educativa que permita:

  • Mantener una temperatura estable (ejemplo: 37.5 °C para huevos de gallina).
  • Aprender a usar un termostato para controlar dispositivos eléctricos.
  • Promover proyectos escolares de biología y electrónica accesibles.

⚙️ Componentes principales

  • Caja de icopor (unicel) aislante
  • Termostato digital
  • Sensor de temperatura
  • Fuente de alimentación
  • Resistencias calefactoras o bombillo incandescente
  • Cableado eléctrico
  • Ventana de observación en acrílico

🔧 Funcionamiento

El termostato mide la temperatura interna con el sensor:

  1. Si la temperatura baja del valor configurado, activa la resistencia calefactora.
  2. Cuando llega al valor deseado, apaga la resistencia para evitar sobrecalentamiento.
  3. Este ciclo se repite constantemente para mantener la incubadora estable.

🛠️ Diseño y construcción

La incubadora fue armada siguiendo estos pasos:

  1. Se preparó la caja de icopor como cámara de incubación.
  2. Se instaló una resistencia calefactora dentro de la caja.
  3. El sensor de temperatura se ubicó en el centro para lecturas precisas.
  4. El termostato se montó en la tapa para fácil acceso.
  5. Se añadió una ventana transparente para observación.
  6. Se probó el sistema regulando la temperatura a 37.5 °C.

📖 Instrucciones de uso

  1. Conectar la incubadora a la corriente.
  2. Configurar el termostato a la temperatura deseada:
    • 🐓 Huevos de gallina: 37.5 °C
    • 🦆 Huevos de pato: 37.2 °C
    • 🐦 Huevos de codorniz: 37.5 °C
  3. Colocar los huevos o material a incubar dentro de la caja.
  4. Revisar periódicamente la temperatura y humedad.

💡 Aplicaciones prácticas

  • Incubación de huevos de diferentes aves.
  • Proyectos escolares en biología.
  • Conservación de microorganismos.
  • Control térmico para experimentos de laboratorio.
